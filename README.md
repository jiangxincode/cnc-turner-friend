# 车工计算 Web 版

将 Android 应用"车工计算"复刻为纯前端单页应用，无需安装，打开即用。

## 功能模块

| 模块 | 说明 |
|------|------|
| 圆弧接点 | 球头/外形/内形接点坐标计算，共 68 个子页面 |
| 各种螺纹 | 公制、英制、美制、T型、蜗杆等 27 种螺纹参数计算 |
| 宏程序 | 钻深孔、车网纹、油槽、去半扣、斜进刀、椭圆宏程序生成 |
| 角度计算 | 锥度角计算、莫氏锥标准参数查询 |
| 公差等级 | IT01–IT18 标准公差数据查询 |
| 对角计算 | 四方、六方、多边形对角距离计算 |
| 材料重量 | 圆棒、方钢、六方、管铝、工字钢等重量计算 |
| 常用代码 | G/M 代码速查、对刀说明 |
| 螺纹对照表 | 公制/英制/美制螺纹对照 |
| 三角函数表 | 0°–90° 正弦、余弦、正切值 |
| 其他计算 | 光洁度换算、线速度、螺纹三针测量、圆等分、中心孔、G98/G99 转换 |
| 工艺品 | 心形、五角星、多边形、子弹、葫芦、陀螺、酒杯、奖杯宏程序生成 |
| 各种槽 | 外圆槽、内孔槽、端面槽加工坐标计算，共 11 个子页面 |
| 解三角形 | 直角/任意三角形求解 |
| 宏转G模拟 | 宏程序解析转换为 G 代码 |
| 实验室 | 切削速度、进给换算、锥度、圆周计算 |

## 本地运行

项目使用 ES Modules，需要通过 HTTP 服务器访问，不能直接双击 `index.html`。

```bash
# 安装依赖（仅首次需要）
npm install

# 启动本地服务器
npm run dev
```

然后打开浏览器访问 `http://localhost:3000`。

## 技术栈

- 纯 HTML + CSS + JavaScript，无任何框架依赖
- ES Modules 模块化组织代码
- Hash 路由实现单页应用导航
- 静态资源从原 APK 提取复用

## 说明

本项目为原 Android 应用"车工计算"的 Web 复刻版，去除了登录、VIP、支付等网络功能，保留全部本地计算功能，所有计算均在客户端完成，不发起任何网络请求。
